#_Proyecto e-commerce_

###_Librerias utilizadas_

*Firebase
*React Router Dom
*React Spinners
*Material UI
*Context (para crear el carrito)

###_Como acceder:_

1. Debemos realizar un npm install
2. Una vez instalado, realizaremos un npm start
